WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the content of a web page. Examples of HTML include headings,paragraphs, images, links, etc

Check out the HTML tutorial on w3schools.com.

Learn CSS

CSS stands for Cascading Styles Sheets. CSS is the style of a webpage. Examples of CSS include front styles and color, backround color, borders,

Check out the CSS tutorial w3schools.com.

Learn JavaScript

JavaScript is the programing language of the web. JavaScript helps to make web pages interactive and dynamic. some common JavaScript are:

  • add new HTML to the page.
  • change the existing content / style,
  • react to user exents (mouse clicks, key presses, etc.)

Check out the JavaScript tutorial on javascript.info.
